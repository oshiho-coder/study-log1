# 模写サイト⑥：アパレルブランド EC商品ページ

## サイト概要
アパレルブランドのECサイトを想定した商品詳細ページを模写制作しました。  
商品画像・商品情報・サイズ選択・サイズ表など、ECサイトに必要な要素を一通り実装しています。

## サイト種類
- ECサイト（商品詳細ページ）

## ページ構成
- 商品ページのみ（1ページ構成）

## 対応端末
- PC
- スマートフォン（レスポンシブ対応）

## 使用技術
- HTML5
- CSS3
- Flexbox
- Grid Layout
- レスポンシブ対応（メディアクエリ）
- Google Fonts
- リセットCSS（ress）

## 実装内容
- Gridを用いた商品画像・商品情報の2カラムレイアウト
- Flexboxによるヘッダー内ナビゲーションの横並び配置
- select要素を用いたサイズ・数量選択UI
- table要素を用いたサイズ表の実装
- 画面幅に応じたレイアウト切り替え（800px / 515px）

## 講師からの指摘
- paddingに `auto` は指定できないため、marginとの役割を整理すること
- CSSが反映されない原因として、CSS読み込み順（head内の記述順）を確認すること
- table内で `th` と `td` の役割を混在させないこと
- レスポンシブ時に要素が重なっている箇所は、不要な指定を整理すること

## 詰まった点
- レイアウト作成時に細部から着手してしまい、全体構造の把握に時間がかかった
- Gridで大枠を分けた後の、各要素の幅調整に試行錯誤した

## 学んだこと
- コーディング前にレイアウト構成を図に起こすことで、  
  wrapper構成やクラス設計に迷いにくくなる
- 見た目の再現だけでなく、HTML構造とCSSの役割分担を意識することが重要である

- JavaScriptを用いたカート処理や動的UIの理解を進める
